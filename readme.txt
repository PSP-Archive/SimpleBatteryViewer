readme
-------------------------------------------------------------------------------------
【はじめに】
HOME画面の日時が表示されているすぐ下にバッテリーの情報を表示します。

-------------------------------------------------------------------------------------
【動作環境】
ALL CFW

-------------------------------------------------------------------------------------
【導入方法】
「SimpleBatteryViewer.prx」をms0:/sepluginsにコピー

http://code.google.com/p/prx-common-libraries/downloads/list

からprxcmlib_pack.zipをダウンロード、解凍して出てきた「lib」フォルダをms0:/sepluginsにコピー

「game.txt」に

ms0:/seplugins/SimpleBatteryViewer.prx 1

と追記
